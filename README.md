# Students Performance Analysis

Week 1 internship project at Maincraft Technologies.

This project performs Exploratory Data Analysis (EDA) on a student performance dataset using Python, Pandas, Matplotlib, and Seaborn.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Objective
Analyze student performance data to identify trends, relationships, and insights through statistical analysis and visualizations.
